🎓 Exam Flow

정기시험 운영에 필요한 다양한 자료들을 출력하고 관리하는 업무 플랫폼

🗂️ 관리실

작업 저장, 백업, 템플릿 및 데이터 초기화를 관리합니다. 입력 내용은 자동으로 이 PC에 저장됩니다.

작업 저장·불러오기

설정 템플릿

시험 규칙·고사실 설정만 저장합니다. 학생 데이터는 포함되지 않습니다.

운영 백업

학생 데이터를 포함한 전체 운영 상태를 백업합니다.

JSON 백업

데이터 초기화

모든 데이터와 저장된 작업을 삭제합니다. 되돌릴 수 없습니다.

🏫 학교 정보

출력물·엑셀 상단에 학교명이 표시됩니다. 학년도·학기는 아래 시험 기본 정보를 따릅니다.

🧭 학년별 이동 방식

👥 학생별 수강 과목

📚 과목별 수강 현황

⚙️ 시험 일정 생성

Step 1 시험 시간표와 Step 2 학생 데이터를 연결하여 학생별 시험 일정과 과목별 응시자 목록을 생성합니다.

📝 교시별 과목 응시자

🏫 교실별 상주 현황

반=교실 원칙으로 본반 학생(이동 제외)과 이동반 유입 학생이 자동 계산됩니다. 과목별 고사실 수동 배정은 필요 없습니다.

🪑 고정 좌석 부여

교실별 상주 학생에게 좌석을 1회 부여합니다. 시험 기간 전체 동일 좌석이 유지되며, 교시별 과목만 연결됩니다.

📋 자료검증

시험실·좌석번호를 최종 확인하고 수정합니다. 변경 사항은 출력물에 반영됩니다.
